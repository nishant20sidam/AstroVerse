# 🌌 AstroVerse

AstroVerse is a simple educational website that provides information about the planets in our solar system. The project is built using **HTML, CSS, and JavaScript** and contains individual pages for each planet with images and basic details.

This project is useful for beginners who want to learn about **frontend web development** and how to structure a multi-page website.

---

## 🚀 Features

- 🌍 Individual pages for each planet
- 📖 Informational content about the solar system
- 🎨 Styled using CSS
- ⚡ Interactive elements using JavaScript
- 📬 Contact page
- 🔐 Login page
- 📱 Simple and clean user interface

---

## 🪐 Planets Included

The website contains separate pages for:

- Mercury
- Venus
- Earth
- Mars
- Jupiter
- Saturn
- Uranus
- Neptune

Each page contains an image and information about the planet.
